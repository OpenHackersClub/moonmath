Distributed systems confront the fundamental tension between autonomy and consistency — how can independent replicas evolve freely yet converge to the same state? These pages explore the event-graph walker (eg-walker) algorithm, which unifies operational transformation and CRDTs into a single framework for collaborative text editing, achieving peer-to-peer convergence with minimal memory overhead.
